# GoMmander

### How to install

```
git clone git@github.com:George-Aidonidis/GoMmander.git
cd GoMmander/
npm install
```

### Run the app

```
npm start
```

### Import to chrome

* Navigate to `chrome://extensions/`
* Enable `Developer mode`
* Click `Load unpacked extension`
* Select the `/build` folder
